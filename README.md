# Vue 3 포트폴리오 레이아웃

Vue 3 + Vite 기반 이직용 웹개발자 포트폴리오 기본 레이아웃입니다.

## 구성 섹션
- 헤더 네비게이션
- 히어로
- 소개
- 기술 스택
- 프로젝트
- 경력
- 연락처
- 푸터

## 실행 방법
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
npm run preview
```
